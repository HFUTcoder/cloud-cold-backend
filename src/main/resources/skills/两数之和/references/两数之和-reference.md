# 两数之和 reference

这个参考文件用于说明“两数之和” skill 的输入和输出约定。

## 输入约定

- 参数 `a`：第一个数字
- 参数 `b`：第二个数字
- 或者使用 `nums` 数组，按顺序提供两个数字，例如 `{"nums":[1,2]}`
- `a` 和 `b` 可以是整数，也可以是小数

## 输出约定

- 返回值为 `a + b` 的结果
- 返回结果优先使用数字本身，不要额外包装无关字段
- 对用户回复时可以直接说“结果是 xx”

## 示例

- `a = 1, b = 2`，结果是 `3`
- `a = 1.5, b = 2.5`，结果是 `4.0`
- `a = -3, b = 8`，结果是 `5`
