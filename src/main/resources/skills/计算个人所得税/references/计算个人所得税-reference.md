# 计算个人所得税 reference

这个参考文件用于说明“计算个人所得税” skill 的输入和输出约定。

## 输入约定

- 参数 `salary`：月度税前工资
- 参数 `insurance`：月度五险一金个人缴纳部分
- 参数 `special_deduction`：月度专项附加扣除
- 参数 `other_deduction`：其他依法扣除项，可选，默认 0
- 以上参数可以是整数，也可以是小数

## 输出约定

- 返回值为一个结构化结果，至少包含：
  - `taxable_income`：应纳税所得额
  - `tax_rate`：适用税率
  - `quick_deduction`：速算扣除数
  - `tax`：应纳个人所得税
- 对用户回复时可以直接说“按当前输入估算，应纳个人所得税为 xx 元”

## 计算规则

- 基本减除费用按月 `5000` 元计算
- 应纳税所得额 = `salary - insurance - special_deduction - other_deduction - 5000`
- 当应纳税所得额小于等于 `0` 时，税额为 `0`
- 本 skill 当前使用简化的月度综合所得税率表：
  - 不超过 `3000`，税率 `3%%`，速算扣除数 `0`
  - 超过 `3000` 至 `12000`，税率 `10%%`，速算扣除数 `210`
  - 超过 `12000` 至 `25000`，税率 `20%%`，速算扣除数 `1410`
  - 超过 `25000` 至 `35000`，税率 `25%%`，速算扣除数 `2660`
  - 超过 `35000` 至 `55000`，税率 `30%%`，速算扣除数 `4410`
  - 超过 `55000` 至 `80000`，税率 `35%%`，速算扣除数 `7160`
  - 超过 `80000`，税率 `45%%`，速算扣除数 `15160`

## 示例

- `salary = 30000, insurance = 3000, special_deduction = 2000, other_deduction = 0`
  结果中应纳税所得额为 `20000`
- `salary = 12000, insurance = 1500, special_deduction = 1000, other_deduction = 0`
  结果中税额应为较低档次
